How to Use

  1. Set the number of players and their names
  2. Set the number of panel rows and columns (e.g., 3√ó3 for 9 panels)
  3. Set the number of balls for each player
  4. Select panel mode (Individual/Shared)
  5. Start the game!
  6. Tap panels to record knocked down panels
  7. If you missed, tap the "Missed" button
  8. After the game ends, rankings are automatically announced
