Frequently Asked Questions (FAQ)

Q: How do I use the app?

A: After launching the app, first set the number of players and their names. Next, set the number of panel rows and columns (e.g., 3×3 for 9 panels). Set the number of balls for each player and select the panel mode, then tap the start game button. During the game, tap on panels you've knocked down to record them.

Q: What's the difference between panel modes?

A: There are two modes:
Individual Mode: Only panels you've knocked down cannot be hit next time. Each player knocks down panels independently
Shared Mode: Once a panel is knocked down, no one can hit it. The game ends when all panels are down

Q: How can I set the number of panels?

A: You can set panels freely by rows and columns. Supports from 3×3 (9 panels) to 10×10 (100 panels). You can change the number of rows and columns in the settings screen.

Q: How many players are supported?

A: Supports 2 to 10 players. You can change the number of players in the settings screen and freely set each player's name.

Q: Can I set the number of balls?

A: Yes, you can set the number of balls for each player freely from 1 to 20. Change "Balls per Player" in the settings screen.

Q: What is Perfect?

A: A player who knocks down all panels. When a perfect is achieved, a special message is displayed, and the player is shown specially in the results screen. If multiple players achieve perfect, the player who used fewer balls ranks higher.

Q: How are rankings determined?

A: Perfect achievers are prioritized first. If there are multiple perfect achievers, they are ranked by the number of balls used. If not perfect, players are ranked by the number of panels knocked down. Players with the same score share the same rank.

Q: Where is data stored?

A: All game data (player names, scores, settings, etc.) is stored locally on your device. It is not transmitted to external servers. When you delete the app, all data will be deleted.

Troubleshooting

  • App won't launch: Try restarting your device and ensure you have the latest version of iOS
  • Panels cannot be tapped: Check if there are remaining balls. Also, in Individual Mode, panels you've already knocked down cannot be tapped
  • Scores not displaying correctly: Try restarting the app. If the problem persists, end the game and start a new game from the settings screen
  • Ads not displaying: Check your internet connection. Also, if you're using an ad blocker, ads may not be displayed
  • App crashes: Try force-closing the app and reopening it. If the problem persists, restart your device
